## ðŸ“Œ KPIs & Insights

To optimize overall call center operations, the following KPIs are monitored:

1. **Increase Call Resolution Rate**  
   - Target: reach **>80% resolution** across all agents  

2. **Reduce Average Speed of Answer**  
   - Goal: reduce waiting time to **<50 seconds**

3. **Improve Customer Satisfaction Score**  
   - Aim for an average rating of **4.0+**

4. **Lower Unanswered Call Percentage**  
   - Target: reduce unanswered calls below **10%**
